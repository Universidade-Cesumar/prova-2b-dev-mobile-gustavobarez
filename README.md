# 📦 Almoxarifado — App de Controle de Estoque

App React Native (Expo) para controle de estoque do laboratório de enfermagem.

## 🚀 Como rodar

```bash
npm install
npx expo start
```

## 🔧 Configurar MockAPI

Abra `src/constants.js` e substitua `SEU_ID_AQUI` pela URL do seu projeto no MockAPI.
